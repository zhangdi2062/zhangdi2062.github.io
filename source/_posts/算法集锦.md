1. 思路

2. 实现
 
代码层面，一般用递归

## 经典题目

### 斐波那契数列

```javascript
function dp(n) {
  if (n === 1 || n === 2) {
    return 1;
  }
  return dp(n - 1) + dp(n -2);
}
```
